# PWA Icons

Ce dossier doit contenir les icônes de l'application pour la PWA.

## Tailles requises :
- `icon-72.png` (72x72)
- `icon-96.png` (96x96)
- `icon-128.png` (128x128)
- `icon-144.png` (144x144)
- `icon-152.png` (152x152)
- `icon-192.png` (192x192)
- `icon-384.png` (384x384)
- `icon-512.png` (512x512)
- `apple-touch-icon.png` (180x180)

## Pour iOS :
- `apple-touch-icon.png` - Icône pour l'écran d'accueil iOS
- `safari-pinned-tab.svg` - Icône SVG monochrome pour Safari

## Raccourcis :
- `shortcut-pos.png` (96x96) - Icône raccourci "Nouvelle Vente"
- `shortcut-stock.png` (96x96) - Icône raccourci "Stock"

## Conseil :
Utilise un outil comme https://realfavicongenerator.net/ pour générer toutes les tailles à partir d'une seule image source (512x512 minimum).
